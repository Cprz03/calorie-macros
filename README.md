# Calorie & Macro Calculator (C++)

A simple CLI app that calculates daily calorie needs and macronutrients.

## Features
- Uses Mifflin-St Jeor Equation
- Supports male and female
- Activity level adjustment
- Goal-based calories (cut, maintain, bulk)
- Macro breakdown:
- Protein (30%)
- Carbs (40%)
- Fats (30%)

## Compile & Run

### Compile:
```bash
g++ main.cpp -o calculator
